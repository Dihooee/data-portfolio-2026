## Key SQL patterns learned
- `SELECT` chooses columns (`*` = all columns)
- `WHERE` filters rows by conditions
- `ORDER BY` sorts results
- `LIMIT` returns top N rows
- `GROUP BY` + `SUM()` aggregates totals by group
- `COUNT(*)` counts rows
